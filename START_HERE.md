## 🔧 How to Add SEO to Other Pages

Use this template for product/brand pages:

```tsx
import SEO from '../components/SEO'

const YourPage = () => {
  return (
    <>
      <SEO 
        title="Your Page Title | Al Emaar Al Areeq Trading Est."
        description="Your page description (150-160 characters)"
        keywords="keyword1, keyword2, keyword3"
        canonical="https://alemaaralareq.com/your-page"
      />
      <div>
        <h1>Your Page Title</h1>
        {/* Your page content */}
      </div>
    </>
  )
}
```

### Example: Sika Brand Page
```tsx
<SEO 
  title="Sika Epoxy Supplier in Jeddah | Sika Products | Al Emaar Al Areeq"
  description="Authorized Sika epoxy supplier in Jeddah. We supply Sika epoxy flooring, waterproofing, repair mortars, bonding agents, construction chemicals."
  keywords="Sika epoxy supplier Jeddah, Sika products Jeddah, Sika flooring Jeddah"
  canonical="https://alemaaralareq.com/sika-products"
/>
```
